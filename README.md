# Chronos

Chronos is a VS Code extension that tracks your coding time and tasks.

## Features
- Tracks active coding sessions ("heartbeats").
- Sends activity data to your Chronos server.
- Provides commands for managing API key and testing connectivity.

## Commands
- `Chronos: Hello` — test command.
- `Chronos: Api Key` — set your API key.
- `Chronos: Test` — check server connection.

## Requirements
- Chronos server (https://next-chronos.vercel.app)
